BigInt get asBigInt

Returns the raw byte value as a BigInt.

Useful for storing in a database as a numeric type (e.g., BIGINT) for improved indexing and query performance.

When converting back to a ResourceId, you must use the ResourceId.fromBigInt constructor and provide the original sizeInBytes to ensure the ID is reconstructed correctly.

Implementation

BigInt get asBigInt {
  final hex = bytes.map((b) => b.toRadixString(16).padLeft(2, '0')).join();
  return BigInt.parse(hex, radix: 16);
}
