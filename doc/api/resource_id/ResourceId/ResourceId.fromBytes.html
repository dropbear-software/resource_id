Reconstructs a ResourceId from its raw bytes.

This is useful when retrieving an ID from a database that stores it as a BINARY or BLOB type.

Implementation

factory ResourceId.fromBytes({
  required String resourceType,
  required Uint8List bytes,
  ResourceId? parent,
}) {
  if (bytes.isEmpty) {
    throw ArgumentError.value(
      bytes,
      'bytes',
      'Cannot be empty. Must contain at least 1 byte.',
    );
  }
  return ResourceId._(
    resourceType: resourceType,
    bytes: bytes,
    parent: parent,
  );
}
